Introduction
==============

Welcome to The Fractal CDT(Contract Development Tools) Documentation, where you can get help to develop your smart contracts on Fractal Platform.   

Fractal Smart Contract is designed upon WebAssembly, which is designed as a portable target for compilation of high-level languages like C/C++/Rust. Fractal Platform now uses C++ for smart contract development.

What can You Learn with This Documentation
------------------------------------------------------
With this document, we can learn: 

	* How to develop smart contracts on Fractal Platform
	* How to deploy and invoke smart contracts on Fractal Platform
